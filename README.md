# Diabetes Tracker

A Flutter application to track diabetes metrics and calculate insulin dosages.

## Features

-   Track blood sugar levels (fasting, breakfast, lunch, dinner).
-   Get insulin dosage recommendations for Lantus and Fiasp.
-   View your history of readings and insulin doses.
-   Export your history as a CSV file.
-   Configure your user settings.

## Installation

1.  Clone the repository.
2.  Install the dependencies: `flutter pub get`
3.  Run the app: `flutter run`

## Usage

1.  Add a new reading by tapping the "+" button on the home page.
2.  Enter your blood sugar level and select the reading type.
3.  The app will recommend an insulin dose if applicable.
4.  You can save the reading and the recommended dose.
5.  View your history on the "History" page.
6.  Export your history from the "History" page.
7.  Configure your settings on the "Settings" page.
